# Next.js CRUD Starter (Auth + Tasks/Notes/Posts/Contacts)

## Setup
1. Install dependencies: `npm install`
2. Create `.env.local` from `.env.example` and set `MONGO_URI` and `JWT_SECRET`.
3. Run: `npm run dev` and open http://localhost:3000

## Notes
- This starter uses Next.js pages router with API routes and MongoDB (Mongoose).
- Deploy to Vercel (set environment variables).
